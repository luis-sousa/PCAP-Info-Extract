# PCAP-Info-Extract
Extração de informação de um ficheiro PCAP, usando Shell Scripting. 

Este trabalho consistiu na elaboração de um script, que analisa uma captura de rede, sob a forma de um ficheiro pcap. O script foi desenvolvido em bash scripting e permite extrair informações do ficheiro pcap. Informações como: listagem de ligações, equipamentos, protocolos, ficheiros presentes, portos utilizados, procurar por emails, ou por qualquer string, criar resumos de ficheiros, entre outras funcionalidades. Além disso, é possível gerar ficheiros de texto e reports em html para apresentar os resultados ao utilizador.

# Executar o script

sudo ./ifc_grupo5.sh captura.pcap pasta_de_saida_resultados
